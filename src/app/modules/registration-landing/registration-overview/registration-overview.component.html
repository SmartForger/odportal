PCTE Registration Overview

By requesting an account for PCTE, you will need to complete the following steps.

Details
Select Account Type

The initial step in the PCTE registration process requires a user to select the account type that they are creating.

Account types include:

  • General User
    General User Accounts include individuals who will be using the PCTE environment, but do not need administrative access.
  • Privileged User (Administrators)
    Privileged User Accounts are named credentials which have been granted administrative privileges on one or more systems.
  • Vendor
    Vendor Accounts are created by entities that supplies services. This account is required when deploying microapplications to PCTE.
Basic Information Collected

Basic information is collected in order to create a PCTE account.

Basic infromation can be collected by entering details directly or importing via CAC/ECA. If a user chooses to enter basic information directly, they will still be required to login with CAC credentials to register.

Login to PCTE

After submitting basic information, a user will be redirected to a login page to enter the PCTE system and continue the registration process.

 Additional User Information Collected

Users will be presented with a "My Registration" microapplication.

The "My Registration" microapplication consists of a wizard that walks users through the required information to register for PCTE.

The first step in the "My Registration" wizard is to compelete an "Additional User Information" form.

The fields of this form will be used to pre-populate the required forms that you will be completing during the registration process..
