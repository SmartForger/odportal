PCTE Registration Overview

By requesting an account for PCTE, you will need to complete the following steps.

Details
Select Account Type

The initial step in the PCTE registration process requires users to select the account type that they wish to create.

Account types include:

  • General User
    General User Accounts include individuals who will be using the PCTE environment, but do not need administrative access.
  • Privileged User (Administrators)
    Privileged User Accounts are named credentials which have been granted administrative privileges on one or more systems.
  • Vendor
    Vendor Accounts are created by entities that supplies services. This account is required when deploying microapplications to PCTE.
Basic Information Collected

Basic information is collected in order to create a PCTE account.

Basic infromation can be collected by entering details directly or importing those details via CAC/ECA. If a user chooses to enter basic information directly, they will still be required to login with CAC credentials prior to registration.

Login to PCTE

After submitting basic information, a user will be redirected to the PCTE login page to continue the registration process.

 Additional User Information Collected

Users will be presented with the "My Registration" microapplication.

The "My Registration" microapplication consists of a wizard that guides users through the different forms that are required to register for PCTE.

The first step in the "My Registration" wizard an "Additional User Information" form.

The details of this form will be used to pre-populate other forms that are completed during the registration process.

 Required User Forms

PCTE registration requires certain forms to be filled out (and approved), prior to having an account created.

General User Forms
  • System Authorization Access Form (SAAR DD2875)
  • Acceptable Use Policy
  • PCTE NDA
Privileged User (Administrators) Forms
  • System Authorization Access Form (SAAR DD2875)
  • Acceptable Use Policy
  • PCTE NDA
  • Privileged Access Agreement
  • Administrator Account Request
Vendor Forms
  • More details coming soon

Form Verification

Each of the completed forms go through a verification process. If a Supervisor, FSO, or IAO have questions regarding the forms, the user will be notified.

 My Certifications/CEUs

Users are required to submit their current CompTIA Security+ certification details.

 Verification & Supporting Items

After uploading the CompTIA Security+ certification details, the registration process is complete.

Administrators, Supervisors, FSOs and IAOs will be reviewing the application details and will notify the user if any issues occur.

Once fully approved, a user will be notified and can access PCTE using the same login credentials that were created at the beginning of the registration process.
